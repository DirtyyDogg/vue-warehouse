# vue-warehouse

[WORK IN PROGRESS] Cross-browser storage for Vue.js

## Build Setup

``` bash
# install dependencies
yarn install

# build for production with minification
yarn run build

# run unit tests
yarn run unit

# run all tests
yarn test
```
